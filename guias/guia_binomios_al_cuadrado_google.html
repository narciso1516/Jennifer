Binomios al cuadrado

Aprende a resolver un binomio al cuadrado sin memorizar de forma mecánica: identificando sus partes, aplicando la regla y cuidando el signo del término de en medio.

1. ¿Qué es un binomio?

Un binomio es una expresión algebraica que tiene dos términos separados por un signo de suma o resta.

x + 5
3a − 2
Idea clave: cuando un binomio está al cuadrado, significa que se multiplica por sí mismo. No significa que solo se eleven los dos términos y ya.

2. Fórmulas principales

Binomio con suma

(a + b)2 = a2 + 2ab + b2

El término de en medio queda positivo.

Binomio con resta

(a − b)2 = a2 − 2ab + b2

El término de en medio queda negativo, pero el último término sigue siendo positivo.

3. Método docente: tres partes

  1. 1
    Eleva el primer término.
    Si el primer término es 3x, entonces (3x)2 = 9x2.
  2. 2
    Calcula el doble producto.
    Multiplica 2 por el primer término y por el segundo término.
  3. 3
    Eleva el segundo término.
    Si el segundo término es 5, entonces 52 = 25.
  4. 4
    Cuida el signo.
    Si el binomio es suma, el término de en medio es positivo. Si es resta, el término de en medio es negativo.

4. Ejemplo 1: (x + 5)2

Paso 1. Identificamos: primer término = x, segundo término = 5.
Paso 2. Como es suma, usamos: (a + b)2 = a2 + 2ab + b2.
(x + 5)2 = x2 + 2(x)(5) + 52
Paso 3. Multiplicamos: 2(x)(5) = 10x y 52 = 25.
(x + 5)2 = x2 + 10x + 25

5. Ejemplo 2: (x − 7)2

Paso 1. Identificamos: primer término = x, segundo término = 7.
Paso 2. Como es resta, usamos: (a − b)2 = a2 − 2ab + b2.
(x − 7)2 = x2 − 2(x)(7) + 72
Paso 3. Multiplicamos: 2(x)(7) = 14x y 72 = 49.
(x − 7)2 = x2 − 14x + 49
Ojo: el último término es positivo porque (−7)(−7) = +49.

6. Ejemplo 3: (3x + 2)2

Paso 1. Primer término = 3x. Segundo término = 2.
(3x + 2)2 = (3x)2 + 2(3x)(2) + 22
Paso 2. Elevamos: (3x)2 = 9x2.
Paso 3. Doble producto: 2(3x)(2) = 12x.
Paso 4. Último cuadrado: 22 = 4.
(3x + 2)2 = 9x2 + 12x + 4

7. Ejemplo 4: (4a − 3)2

Paso 1. Primer término = 4a. Segundo término = 3.
(4a − 3)2 = (4a)2 − 2(4a)(3) + 32
Paso 2. Elevamos: (4a)2 = 16a2.
Paso 3. Doble producto: 2(4a)(3) = 24a.
Paso 4. Último cuadrado: 32 = 9.
(4a − 3)2 = 16a2 − 24a + 9

9. Error común

Error:

(x + 4)2 = x2 + 16

Por qué está mal: faltó el doble producto: 2(x)(4) = 8x.

Correcto:

x2 + 8x + 16

Resumen final

Un binomio al cuadrado siempre tiene tres partes: cuadrado del primero, doble producto y cuadrado del segundo. El signo del término de en medio depende del signo que esté dentro del paréntesis.
