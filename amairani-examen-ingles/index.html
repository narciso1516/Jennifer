Método rápido para contestar

1
Busca la palabra pista.Yesterday = pasado. Tomorrow = futuro. Every day = rutina. If = condicional.
2
Identifica el sujeto.Con he, she, it, en presente simple el verbo normalmente lleva -s.
3
Pregunta qué quiere decir la oración.Obligación = must / have to. Consejo = should. Imaginario = would.
4
No traduzcas palabra por palabra.Primero encuentra el tiempo verbal y luego elige la opción correcta.

Acordeón de temas del examen

Cuándo aparece

Se usa para rutinas, hábitos o cosas que pasan normalmente.

Subject + verb
She / He / It + verb + s
Ejemplo: She goes to school every day.
Pista: every day, always, usually, sometimes.
Error común: She go ❌ · She goes ✅

Se usa cuando la acción ya terminó.

Regular: play → played
Irregular: go → went
Ejemplo: Yesterday, I visited my grandmother.
Pista: yesterday, last week, last year, ago.

Must y have to se usan cuando algo es obligatorio.

You must study.
You have to wear a uniform.
Pista: si suena a “tienes que” o “debes”, piensa en must / have to.

Should y ought to sirven para dar consejos.

You should rest.
You ought to study more.
Ejemplo: You have a headache. You should rest.

Will se usa para futuro. Would se usa para situaciones imaginarias.

I will study tomorrow.
If I had money, I would travel.
Pista: tomorrow = will. If I had / If I were = would.
Zero: If + present, present
If you heat water, it boils.
First: If + present, will + verb
If I study, I will pass.
Second: If + past, would + verb
If I had money, I would travel.

Se usa para experiencias o acciones conectadas con el presente.

Subject + have / has + past participle
Ejemplo: She has visited Paris.
Pistas: already, yet, just, recently, ever, never.

Since marca el punto de inicio. For marca duración.

since 2020 · since Monday
for two years · for three days
This = singular cerca
That = singular lejos
These = plural cerca
Those = plural lejos
Ejemplo: These are my books.

Se usa cuando lo importante es la acción, no quién la hace.

Present: am / is / are + participle
Past: was / were + participle
Ejemplo: English is spoken in many countries.
