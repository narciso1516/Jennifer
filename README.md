# Experiencia web para Jennifer

Sitio web personal con una invitación interactiva y recursos creados para experiencias educativas.

## Tecnología

- HTML
- CSS
- JavaScript
- GitHub Pages
- Diseño mobile-first

## Uso

Abre `index.html` en un navegador.

## Privacidad y alcance

El repositorio es público y está relacionado con una persona. Antes de mantenerlo publicado, revisa que no contenga información privada, ubicaciones, teléfonos o datos de menores.

El historial incluye ejercicios académicos además de la invitación. Para mantener una estructura clara, esos materiales deberían vivir en un repositorio educativo independiente.
