Factorización para Juan

3 tipos principales con 6 ejemplos resueltos paso a paso.

Objetivo

Juan aprenderá a reconocer y resolver tres tipos de factorización: factor común, diferencia de cuadrados y trinomios donde se buscan dos números que sumen y multipliquen.

La regla más importante es esta: antes de resolver, primero se identifica el tipo de factorización.

Los 3 tipos

Tipo 1

Factor común

ab + ac = a(b + c)

Se usa cuando todos los términos tienen algo repetido.

Tipo 2

Diferencia de cuadrados

a² − b² = (a + b)(a − b)

Se usa cuando hay dos cuadrados perfectos separados por una resta.

Tipo 3

Dos números

x² + bx + c = (x + m)(x + n)

Se buscan dos números que sumen b y multipliquen c.

Tipo 1: Factor común

El factor común es el número, letra o combinación que aparece en todos los términos. Para factorizar, se saca ese factor afuera del paréntesis.

Juan debe preguntar: ¿qué tienen en común todos los términos?
Ejemplo 1

Factoriza:

6x + 18
  1. Observamos los términos:
    6x
    18
  2. Buscamos el número que divide a ambos. El número 6 divide a los dos:
    6x ÷ 6 = x
    18 ÷ 6 = 3
  3. Sacamos el 6 como factor común:
    6x + 18 = 6(x + 3)
  4. Verificamos:
    6(x + 3) = 6x + 18
Resultado: 6x + 18 = 6(x + 3)
Ejemplo 2

Factoriza:

12x² + 8x
  1. Observamos los términos:
    12x²
    8x
  2. Buscamos el mayor número que divide a 12 y 8:
    4
  3. Revisamos la letra. Ambos términos tienen:
    x
  4. Entonces el factor común completo es:
    4x
  5. Dividimos cada término entre 4x:
    12x² ÷ 4x = 3x
    8x ÷ 4x = 2
  6. Escribimos la factorización:
    12x² + 8x = 4x(3x + 2)
Resultado: 12x² + 8x = 4x(3x + 2)

Tipo 2: Diferencia de cuadrados

Una diferencia de cuadrados tiene dos términos, una resta y ambos términos son cuadrados perfectos.

a² − b² = (a + b)(a − b)
  • Debe haber dos términos.
  • Debe haber signo menos.
  • Ambos términos deben tener raíz cuadrada exacta.
Ejemplo 3

Factoriza:

x² − 25
  1. Revisamos si hay dos términos:
    x²
    25
  2. Hay una resta:
    x² − 25
  3. Revisamos si ambos son cuadrados perfectos:
    x² = (x)²
    25 = 5²
  4. Identificamos:
    a = x
    b = 5
  5. Aplicamos la fórmula:
    x² − 25 = (x + 5)(x − 5)
Resultado: x² − 25 = (x + 5)(x − 5)
Ejemplo 4

Factoriza:

9x² − 49
  1. Revisamos los términos:
    9x²
    49
  2. Hay una resta:
    9x² − 49
  3. Revisamos los cuadrados perfectos:
    9x² = (3x)²
    49 = 7²
  4. Identificamos:
    a = 3x
    b = 7
  5. Sustituimos:
    9x² − 49 = (3x + 7)(3x − 7)
Resultado: 9x² − 49 = (3x + 7)(3x − 7)
Esta fórmula no se usa con suma. Por ejemplo:
x² + 25
no se factoriza como diferencia de cuadrados.

Tipo 3: Dos números que sumen y multipliquen

Este método se usa cuando el trinomio empieza con:

x²

La forma general es:

x² + bx + c

Juan debe encontrar dos números que cumplan:

m + n = b
m · n = c
Ejemplo 5

Factoriza:

x² + 7x + 12
  1. Identificamos los valores:
    b = 7
    c = 12
  2. Buscamos dos números que sumen 7 y multipliquen 12:
    3 + 4 = 7
    3 · 4 = 12
  3. Los números son:
    3 y 4
  4. Escribimos los factores:
    (x + 3)(x + 4)
  5. Verificamos:
    (x + 3)(x + 4) = x² + 4x + 3x + 12
    = x² + 7x + 12
Resultado: x² + 7x + 12 = (x + 3)(x + 4)
Ejemplo 6

Factoriza:

x² − x − 12
  1. Identificamos los valores:
    b = −1
    c = −12
  2. Necesitamos dos números que sumen −1 y multipliquen −12.
  3. Probamos con 3 y −4:
    3 + (−4) = −1
    3 · (−4) = −12
  4. Los números son:
    3 y −4
  5. Escribimos los factores:
    (x + 3)(x − 4)
  6. Verificamos:
    (x + 3)(x − 4) = x² − 4x + 3x − 12
    = x² − x − 12
Resultado: x² − x − 12 = (x + 3)(x − 4)
